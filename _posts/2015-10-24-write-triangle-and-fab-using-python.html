用python写斐波那契数列和杨辉三角

斐波那契数列： 

def fab(max):
	n,a,b = 0,0,1
	while n<max:
		yield b
		a,b = b,a+b
		n = n+1

调用：

g = fab(10)
for x in g:
    print x

杨辉三角： 

def triangles(max):
	n,a,b = 0,[1],[]

	while n < max:
		yield a
		a.insert(0,0)
		a.append(0)
		length = len(a)
		for index,value in enumerate(a):
			if index < length - 1:
				b.append(value+a[index+1])

		a,b = b,[]
		n = n+1

调用：

g = triangles(10)
for x in g:
	print x
