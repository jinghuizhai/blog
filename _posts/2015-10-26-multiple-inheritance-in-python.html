pyhton的多重继承

java和php中并没有多重继承的机制，而python的类可以同时继承多个类，选择组合不同类的功能，就可以快速构造出所需要的子类。这称为MixIn（不是迷信）。

内燃机汽车类： 

class combustion_car(object):

	def add_oil(self):
		print 'give me some oil'

电动汽车类： 

class electric_car(object):

	def charge(self):
		print 'give me some electrcity'

混合动力汽车类：

class hybird_car(combustion_car,electric_car):
	pass

实例： 

toyota = hybird_car()
toyota.add_oil() # give me some oil
toyota.charge()  # give me some electricity
